# Screenshots — Instruções

Adicione os prints do sistema nesta pasta com os nomes exatos abaixo.  
Resolução recomendada: **1280 × 800 px** (ou superior), formato **PNG**.

| Arquivo | O que capturar |
|---|---|
| `student-dashboard.png` | Dashboard do aluno logado como Lucas — exibe atividades pendentes, média geral e progresso por disciplina |
| `teacher-dashboard.png` | Dashboard do professor logado como Ana Lima — exibe fila de correções, turmas ativas e KPIs |
| `coordinator-dashboard.png` | Dashboard do coordenador — exibe gráfico de distribuição de notas, evolução mensal e alunos em risco |
| `guardian-dashboard.png` | Painel do responsável — exibe boletim do filho, frequência e lista de atividades |
| `admin-dashboard.png` | Painel administrativo — exibe lista de escolas, KPIs consolidados e detalhes de uma escola |

## Dicas para bons prints

- Use o perfil demo correspondente (ver README principal para credenciais)
- Prefira capturar com a barra lateral expandida para mostrar a navegação
- No coordenador, deixe os gráficos visíveis antes de capturar
- Resolução 1280 × 800 produz imagens nítidas no GitHub e no Codecanyon
